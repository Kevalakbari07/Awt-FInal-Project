## 📋 PROJECT OVERVIEW

### Purpose
A comprehensive web-based **Dairy Management System** that helps farmers and administrators manage milk collection, payments, farmer information, and government schemes for dairy farming.

### Key Features
- 👥 **User Authentication** - Login system with role-based access
- 📊 **Dashboard** - Real-time analytics and milk collection charts
- 👨‍🌾 **Farmer Management** - Track and manage farmer information
- 🥛 **Milk Collection** - Record daily milk collection from farmers
- 💰 **Payment Management** - Process and approve farmer payments
- 📈 **Reports** - Generate and view detailed reports
- 🎁 **Schemes** - Showcase government dairy assistance schemes
- 🔐 **Admin Panel** - Protected admin routes with dashboard, farmer management, payment approval, reports, and schemes management

### User Roles
1. **Regular User** - Access to dashboard, farmers, milk collection, payments, reports, and schemes viewer
2. **Admin** - Full access to admin panel with management capabilities

## 🔐 AUTHENTICATION & AUTHORIZATION

### Login System (`src/pages/Login.tsx`)
- **Credentials Storage:** localStorage
- **Validation Rules:**
  - Username: Minimum 4 characters required
  - Password: Minimum 6 characters + 1 uppercase letter + 1 number
- **localStorage Keys:**
  - `role` - User role ("admin" or "user")
  - Additional user data stored as needed

### Admin Route Protection (`src/components/AdminProtectedRoute.tsx`)
```tsx
- Checks: localStorage.getItem("role") === "admin"
- Non-admin redirect: Navigate to "/" (login page)
- Protected routes:
  ✓ /admin-dashboard
  ✓ /manage-farmers
  ✓ /manage-payments
  ✓ /manage-reports
  ✓ /manage-schemes
```

### Logout Functionality
- **Location:** `Header.tsx` - Logout button
- **Action:** Clears all localStorage data
- **Redirect:** Returns to login page

```
User Access /admin-dashboard
    ↓
AdminProtectedRoute Check
    ↓
localStorage.getItem("role")
```
